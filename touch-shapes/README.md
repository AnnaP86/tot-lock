Touch Shapes

Open `index.html` in a browser (double-click or via a simple HTTP server).

Quick server (Python 3):

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser and open the `touch-shapes` folder.

Tap/click anywhere to spawn random colored shapes. Multiple touches spawn multiple shapes.

Files:
- `index.html` — app shell
- `styles.css` — styles and animations
- `script.js` — touch/click logic
